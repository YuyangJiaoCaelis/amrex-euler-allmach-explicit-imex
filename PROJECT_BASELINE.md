# Project Baseline

This repository is the clean AMReX Euler research baseline for continued development.

It contains the current structured implementation of:

- Explicit O2 HLLC
- Explicit O2 Low-Mach Corrected HLLC-P
- IMEX T1/S2 BDLTV20
- reproduction scripts for Riemann, Gresho vortex, periodic advection blob, and same-gamma shock-density-bubble tests

Use this repository as the starting point for future research tasks, including hardware timing, MPI/CUDA preparation, solver refinement, and additional validation.

New code development should start from this baseline.
